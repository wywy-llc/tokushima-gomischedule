# tokushima-gomischedule

徳島のゴミの日カレンダーを生成する GAS です

## ご利用方法

1. 以下の徳島市のサイトからお住まいの地域が A から D 地区のどこなのか確認する
2. 自分の地区にあった Google カレンダーを選択する
3. カレンダー右下の「+」ボタンをクリックしてカレンダー追加する
